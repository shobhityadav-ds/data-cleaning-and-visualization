## 📖 Project Overview

This project demonstrates an end-to-end **Data Cleaning and Data Visualization** workflow using the **Supermarket Sales Dataset**.

The dataset was analyzed using Python to identify sales trends, customer behavior, and business performance. The project showcases practical data preprocessing techniques and visualization skills commonly used in data analytics.

---

## 🎯 Objectives

- Clean and preprocess raw data
- Handle missing values and duplicate records
- Create informative visualizations

---

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/shobhityadav-ds/data-cleaning-and-visualization.git
```

2. Install the required libraries

```bash
pip install -r requirements.txt
```

3. Open the notebook

```text
Data_cleaning_&_Visualization_Project.ipynb
```

4. Run all cells.
